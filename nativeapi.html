Kiosk Native Interface

These are the functions avialable to any web page loaded through this application that allows access to native system functionality. The javascript is automatically loaded every time a new page is loaded and can be accessed using the kiosk qualifier.

							
kiosk.lock(function(response) {
  if (response["locked"] == true) {
    console.log('Device is now in kiosk mode.')
  }
});

// New approach
kiosk.lock(function(isLocked) {
   if (isLocked) {
      console.log('Device is now in kiosk mode.')
   }
})

kiosk.lock(callback(isLocked))

This call locks down the device to provide a secure testing environment. The steps needed to lock down a device will be specific to operating system being used, but generally this method should prevent access to any other applications, disable any possible interuptions, and ensure that the student can not escape the application.

The success of this call can be determined by passing in a callback function that accepts a "response" parameter. The response will contain a JSON object with a boolean "locked" to signify the success of the call.

Results

{
  "locked": true,
}

kiosk.unlock(callback(isLocked))

Unlocks the device so that the user can easily exit the application. Any content displayed to the user while in an unlocked state should be considered unsecured.

The success of this call can be determined by passing in a callback function that accepts a "response" parameter. The response will contain a JSON object with a boolean "locked" to signify the success of the call.

Results

{
   "locked": false,
}

kiosk.isLocked(callback(isLocked))

Gets the status of the devices kiosk mode. Check the responses locked value to determine if the device is currently in our out of locked mode.

Results

{
   "locked": false,
}

kiosk.getVolume(callback(level))

Gets the system volume level. Range will be between 0.0 and 1.0, where 0.0 means the device is muted.

Results

{
  "level": 0.35
}

kiosk.setVolume(level, callback(err))

Sets the system volume level. Level should be between 0 and 1, where setting to 0.0 will mute the device.

Parameter Value Description
level Volume level to set the system volume to.

Results

{
            "level": 0.35
            }

kiosk.clearClipboard()

Clears the system clipboard so that no data can be pasted to an open ended item. Call this method at the begining and end of a test and before and after any questions with text inputs.

kiosk.onBreach(callback())

Registers a callback that will be called any time the application detects that kiosk mode has been unexpectedly lost.

kiosk.exit(saveState, restart)

Closes the application with options to save the testers current state and to automatically restart.

Parameter Value Description
saveState Set to true if the tester's current progress (cookies and url) should be saved before exiting.
restart Set to true if application should be restarted after it closes.

kiosk.storage.set(key, object, callback(success))

Encrypts and stores an object on the local disk. Sends true to the callback if object wass successfully saved, otherwise false.

Parameter Value Description
key Unique identifier of object being stored.
object Object to be stored. This can be any format.

Results

kiosk.storage.get(key, callback(object))

Returns the decrypted object associated with the provided key. If not found or if there were any issues, the returned object will be null.

Parameter Value Description
key Unique identifier of object being returned.

Results

kiosk.storage.keys(callback(list))

Returns an array of the unique keys currently being stored.


Results

kiosk.storage.remove(key, callback(success))

Removes the object associated with the given key. Passes true back in the callback if remove was successful, otherwise false.

Parameter Value Description
key Unique identifier of object being removed.

Results

kiosk.storage.clear(callback(success))

Removes all objects stored on the local disk. Passes true back in the callback if clear was successful, otherwise false.


Results

kiosk.browser.gotoUrl(url)

Navigates the browser to the given url.

Parameter Value Description
url Url to navigate to.

kiosk.browser.reload()

A hard reload of the current url. This will clear any cached content before reloading the page.

kiosk.browser.clearCache()

Clears the browsers cache so that all future pages are completely reloaded with content from the server.

kiosk.browser.clearCookies()

Clears any cookies that are currently stored by the browser.
