Kiosked API Test Page

Promise-based API methods on the kiosked object. Each card verifies the method exists on load. Click Run to execute a method and see the results.

kiosked.isLocked()
●

Returns true if the device is locked, otherwise false.


    

  


  
  

    

      kiosked.getLockedStatus()
      

        
        ●
      

    

    

      
Returns detailed lock state info: locked, focused, aacMode.

      
      

    

  


  
  

    

      kiosked.lock()
      

        
        ●
      

    

    

      
Activates lockdown mode. Throws if it fails to secure the device.

      
      

    

  


  
  

    

      kiosked.unlock()
      

        
        ●
      

    

    

      
Deactivates lockdown mode. Throws if it fails to unlock.

      
      

    

  


  
  

    

      kiosked.getDeviceInfo()
      

        
        ●
      

    

    

      
Returns hardware, OS, and application details.

      
      

    

  


  
  

    

      kiosked.getProcessList()
      

        
        ●
      

    

    

      
Returns an array of running process names.

      
      

    

  


  
  

    

      kiosked.exit()
      

        
        ●
      

    

    

      
Unlocks the device and closes the application.

      
      

    

  


  
  

    

      kiosked.addEventListener(type, listener)
      ●
    

    

      
Registers an event listener for application events. Events: breach, lost-focus, display-added, offline, online.

    

  


  
  

    

      kiosked.removeEventListener(type, listener)
      ●
    

    

      
Removes a previously registered event listener.
