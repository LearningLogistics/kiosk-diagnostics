Target Blank Test

Verify that links with target="_blank" are blocked in lockdown mode. Select the test link below and try each shortcut. None of these should open the link.

https://www.google.com
Test Steps
  1. 1. Click the link — should not open in a new tab or window.
  2. 2.
    Select the link (click it or tab to it), then try each shortcut:
    • Enter — opens link in current tab
    • Ctrl+Enter — opens in new background tab
    • Ctrl+Shift+Enter — opens in new foreground tab
    • Shift+Enter — opens in new window
  3. 3. Right-click the link — context menu options to open in new tab/window should be blocked or have no effect.

All of the above should be prevented in lockdown mode.
